# SpeedX Car Game — Fix for "The package could not be installed"

This WordPress error usually happens when the **wrong ZIP** is uploaded.

## ✅ Upload the correct ZIP only
Do **NOT** upload:
- the repository ZIP (for example: `shahzain-game-plugin-main.zip`)
- any ZIP where plugin files are nested too deep

Do upload:
- `dist/speedx-car-game.zip` (generated by the build script below)

## 1) Build the installable plugin package
From this project folder run:

```bash
./build-plugin-zip.sh
```

It will generate:

- `dist/speedx-car-game.zip`

## 2) Install in WordPress Admin
1. Go to **Plugins → Add New → Upload Plugin**
2. Select `dist/speedx-car-game.zip`
3. Click **Install Now**
4. Click **Activate**

## 3) If hosting blocks ZIP upload
Manual install via FTP/File Manager:
1. Extract `dist/speedx-car-game.zip`
2. Upload `speedx-car-game/` into `wp-content/plugins/`
3. Activate **SpeedX Car Game** in WordPress Plugins page

## Shortcode
Use this on any page/post:

```text
[speedx_car_game]
```
