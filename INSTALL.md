# SpeedX Car Game Plugin (Ready)

You asked for the **plugin itself**. This repository already contains the plugin code in:

- `speedx-car-game/`

Main plugin file:

- `speedx-car-game/speedx-car-game.php`

## Use it in WordPress
1. Run:
   ```bash
   ./build-plugin-zip.sh
   ```
2. Upload this exact file in WordPress:
   - `dist/speedx-car-game.zip`
3. Activate plugin.
4. Add shortcode:
   ```text
   [speedx_car_game]
   ```

## Important
Do not upload repository source ZIPs such as `shahzain-game-plugin-main.zip`.
Upload only `dist/speedx-car-game.zip`.
